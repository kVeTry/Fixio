# Fixio 💪

App Android para registrar dolencias físicas, recibir consejos y ejercicios personalizados, y ver tu progreso.

⚠️ Fixio no reemplaza el consejo médico profesional.

## Cómo compilar el APK (GitHub)
1. Crea un repositorio nuevo en GitHub.
2. Sube TODO el contenido de este ZIP (arrastra los archivos, incluyendo la carpeta `.github`).
3. Ve a la pestaña **Actions** → workflow **Build APK** → **Run workflow** (o se lanza solo al subir).
4. Cuando termine, descarga el artefacto **Fixio-APK** → dentro está `app-debug.apk`.
5. Instálalo en tu Android (permite "orígenes desconocidos").

## Funciones
- Cuerpo interactivo (frente/espalda, masculino/femenino) con 30+ zonas y dolor 1-10 por zona con notas.
- Análisis diario: motor local de consejos, o IA real (Claude) si añades tu clave API en Ajustes.
- Ejercicios detallados paso a paso por zona, con duración y precauciones.
- Progreso: gráficas de dolor medio, zonas por día, zonas más frecuentes y tendencia semanal.
- Racha diaria 🔥, ánimo/sueño/actividad, consejo del día.
- Notificación diaria configurable para recordarte el registro.
- Temas: Auto, Claro, Oscuro y AMOLED.
- Exportar historial en CSV.
